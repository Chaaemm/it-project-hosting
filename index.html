Cloud Web Hosting (GitHub)

การอัปโหลดเว็บไซต์ขึ้นใช้งานผ่าน GitHub ตามเอกสารทางการ
เสนอ: อาจารย์ศุภชัย พุทธรักษ์
อาจารย์ที่ปรึกษาโครงการ

ขั้นตอนการอัปโหลดเว็บไซต์ขึ้น GitHub

1. สมัครสมาชิก GitHub
เข้าไปที่เว็บไซต์ https://github.com แล้วทำการสมัครสมาชิกโดยกรอกอีเมล ชื่อผู้ใช้ และรหัสผ่าน จากนั้นยืนยันตัวตนให้เรียบร้อย
2. สร้าง Repository (พื้นที่เก็บโปรเจค)
หลังจากเข้าสู่ระบบ ให้กดปุ่ม “New” หรือ “Create Repository” จากนั้นตั้งชื่อโปรเจค เลือกเป็น Public และกด Create Repository
3. อัปโหลดไฟล์เว็บไซต์
ภายใน Repository ให้กดปุ่ม “Add file” → “Upload files” จากนั้นเลือกไฟล์เว็บไซต์ เช่น HTML, CSS, JavaScript แล้วกด Commit changes
4. เปิดใช้งาน GitHub Pages
ไปที่ Settings → Pages → เลือก Branch เป็น main และกด Save ระบบจะสร้างลิงก์เว็บไซต์ให้โดยอัตโนมัติ
5. เข้าใช้งานเว็บไซต์
หลังจากเปิด GitHub Pages แล้ว จะได้ลิงก์เว็บไซต์ เช่น https://username.github.io/repository-name ซึ่งสามารถนำไปใช้งานได้ทันที
หมายเหตุ
- GitHub Pages รองรับเว็บไซต์แบบ Static (HTML, CSS, JavaScript)
- ไม่รองรับ PHP หรือฐานข้อมูล
- เหมาะสำหรับเว็บไซต์แนะนำผลงาน หรือโปรเจคทั่วไป
