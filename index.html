Cloud Web Hosting

รายงานการนำเว็บไซต์ขึ้นใช้งานบนระบบคลาวด์ด้วย GitHub
เสนอ: อาจารย์ศุภชัย พุทธรักษ์
อาจารย์ที่ปรึกษาโครงการ

1. ความหมายของ Web Hosting บนคลาวด์

Web Hosting บนคลาวด์ (Cloud Web Hosting) คือ การนำเว็บไซต์ไปจัดเก็บและให้บริการผ่านระบบเซิร์ฟเวอร์บนอินเทอร์เน็ต โดยใช้เทคโนโลยีคลาวด์ ซึ่งมีการกระจายข้อมูลไปยังหลายเซิร์ฟเวอร์ ทำให้เว็บไซต์สามารถเข้าถึงได้ตลอดเวลา มีความเสถียรสูง และรองรับผู้ใช้งานจำนวนมากได้ดีกว่าการใช้เซิร์ฟเวอร์แบบเดิม นอกจากนี้ยังสามารถขยายทรัพยากรได้ตามความต้องการ เช่น เพิ่มพื้นที่จัดเก็บข้อมูลหรือเพิ่มความเร็วในการประมวลผล

2. เครื่องมือที่ใช้ในการพัฒนาและอัปโหลดเว็บไซต์

GitHub เป็นแพลตฟอร์มสำหรับจัดเก็บโค้ดและควบคุมเวอร์ชัน (Version Control) โดยใช้ระบบ Git ผู้พัฒนาสามารถอัปโหลดไฟล์เว็บไซต์ เช่น HTML, CSS และ JavaScript ขึ้นไปเก็บไว้บน GitHub ได้ นอกจากนี้ยังมีบริการ GitHub Pages ที่ช่วยให้สามารถนำเว็บไซต์ขึ้นออนไลน์ได้ฟรี เหมาะสำหรับเว็บไซต์แบบ Static เช่น เว็บไซต์แนะนำผลงาน หรือเว็บไซต์โปรเจค

3. ขั้นตอนการอัปโหลดเว็บไซต์ขึ้น GitHub

1. สมัครสมาชิก GitHub
เข้าไปที่ https://github.com และสมัครสมาชิกให้เรียบร้อย

2. สร้าง Repository
กดปุ่ม New → ตั้งชื่อโปรเจค → เลือก Public → Create Repository

3. อัปโหลดไฟล์เว็บไซต์
กด Add file → Upload files → เลือกไฟล์ → Commit changes

4. เปิดใช้งาน GitHub Pages
ไปที่ Settings → Pages → เลือก Branch เป็น main → Save

5. เข้าใช้งานเว็บไซต์
ระบบจะสร้างลิงก์เว็บไซต์ เช่น https://username.github.io/project

หมายเหตุ:
รองรับเฉพาะ HTML, CSS, JavaScript ไม่รองรับ PHP
