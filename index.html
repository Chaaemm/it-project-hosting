Cloud Web Hosting

การนำเว็บไซต์ขึ้นใช้งานบนระบบคลาวด์ด้วย GitHub และ Vercel
เสนอ: อาจารย์ศุภชัย พุทธรักษ์
อาจารย์ที่ปรึกษาโครงการ

ขั้นตอนการอัปโหลดเว็บไซต์ขึ้นโฮสติ้ง

1. สมัคร GitHub
เข้าเว็บไซต์ https://github.com แล้วสมัครสมาชิกให้เรียบร้อย
2. สร้าง Repository
กด New Repository ตั้งชื่อโปรเจค และเลือก Public
3. อัปโหลดไฟล์
กด Upload files และเพิ่มไฟล์เว็บไซต์ของคุณ
4. สมัคร Vercel
เข้า https://vercel.com และสมัครด้วย GitHub
5. Deploy
กด New Project → เลือก Repository → Deploy
หมายเหตุ
- Vercel ใช้กับเว็บ HTML ได้ทันที
- ถ้าเป็น PHP ต้องใช้โฮสอื่น
