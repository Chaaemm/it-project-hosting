Cloud Web Hosting

รายงานการศึกษาการนำเว็บไซต์ขึ้นใช้งานบนระบบคลาวด์
เสนอ: อาจารย์ศุภชัย พุทธรักษ์
อาจารย์ที่ปรึกษาโครงการ

วิธีการอัปโหลดเว็บไซต์ขึ้นโฮสติ้ง (GitHub + Vercel)

ขั้นตอนที่ 1: สมัคร GitHub
เข้าเว็บไซต์ GitHub → กด Sign up → กรอกอีเมล รหัสผ่าน และชื่อผู้ใช้
ขั้นตอนที่ 2: สร้าง Repository
กดปุ่ม New → ตั้งชื่อโปรเจค → เลือก Public → Create Repository
ขั้นตอนที่ 3: อัปโหลดไฟล์เว็บไซต์
กด Upload files → ลากไฟล์ HTML, CSS, JS ลงไป → Commit changes
ขั้นตอนที่ 4: สมัคร Vercel
เข้าเว็บไซต์ Vercel → สมัครโดยใช้ GitHub เพื่อเชื่อมต่ออัตโนมัติ
ขั้นตอนที่ 5: เชื่อมโปรเจค
กด Add New Project → เลือก Repository → Import
ขั้นตอนที่ 6: Deploy เว็บไซต์
กด Deploy → รอระบบ → จะได้ลิงก์เว็บไซต์พร้อมใช้งานทันที
หมายเหตุ:
- Vercel รองรับ HTML, CSS, JavaScript
- หากใช้ PHP ต้องใช้โฮสอื่น เช่น InfinityFree หรือ 000webhost

วิดีโอสอนการ Deploy เว็บไซต์
